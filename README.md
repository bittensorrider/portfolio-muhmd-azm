# portfolio-muhmd-azm

portfolio website built with **Vite + React + Tailwind**.  
Content is powered by a simple `data/data.json` file.

## Prerequisites

```yaml
node.js: 20.20.2
```

## Getting Started 🚀

```bash
npm i

npm run dev
# http://localhost:5173

npm run build

npm run preview
```
